# Exames

## Solicitar Exame
O veterinário solicita exames durante a consulta. Você não precisa solicitar pelo portal — a solicitação é feita pelo profissional.

## Acompanhar Exames

1. Acesse **Exames** no menu
2. Você vê todos os exames solicitados para seus pets
3. Cada exame mostra:
   - **Pet**
   - **Tipo de exame** (hemograma, bioquímico, ultrassom, raio-X)
   - **Data da solicitação**
   - **Status**: Solicitado, Coletado, Em análise, Concluído
   - **Veterinário** solicitante

## Status dos Exames

| Status | Significado |
|--------|-------------|
| **Solicitado** | O veterinário pediu o exame |
| **Coletado** | A amostra foi coletada |
| **Em análise** | O laboratório está processando |
| **Concluído** | Resultado disponível! |

## Ver Resultados

1. Quando o exame estiver **Concluído**, clique para abrir
2. Você verá:
   - **Resultado completo** com valores e referências
   - **Laudo** do veterinário (se houver)
   - **Arquivos** anexados (imagens, PDFs)
3. Os resultados ficam salvos no histórico do pet

## Tipos de Exame

### Laboratoriais
- **Hemograma**: Avalia glóbulos vermelhos, brancos e plaquetas
- **Bioquímico**: Função renal, hepática, glicemia, lipídios
- **Urinálise**: Avaliação da urina
- **Parasitológico**: Pesquisa de parasitas
- **Sorologias**: Testes para doenças específicas

### Imagem
- **Radiografia (Raio-X)**: Ossos, tórax, abdome
- **Ultrassom**: Órgãos internos, gestação
- **Ecocardiograma**: Coração
- **Tomografia**: Exames mais detalhados

### Outros
- **Testes rápidos**: Cinomose, parvovirose, FIV/FeLV, leishmaniose
- **Citologia**: Análise de células
- **Biópsia**: Análise de tecidos

## Dúvidas sobre Resultados?

- Agende um retorno para discutir os resultados com o veterinário
- Você pode enviar os resultados pelo chat para a clínica
- Resultados alterados podem exigir novos exames ou tratamento

## Dicas

- Siga corretamente as instruções de **jejum** e **preparo**
- Resultados de exames de imagem (raios-X, ultrassom) são entregues em arquivo digital
- Guarde os resultados para comparação futura
