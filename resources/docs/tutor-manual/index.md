# Manual do Tutor

Bem-vindo ao **Portal do Tutor** VetEssence! Este manual explica tudo que você pode fazer pelo portal.

## Acessar o Portal

1. Abra o navegador e acesse o link fornecido pela clínica
2. Clique em **Entrar** (ou **Cadastre-se** se for seu primeiro acesso)
3. Informe seu **e-mail** e **senha**
4. Pronto! Você está no **Dashboard**

## O que você pode fazer?

| Seção | Descrição |
|-------|-----------|
| **Dashboard** | Visão geral com seus pets, consultas e faturas |
| **Meus Pets** | Dados, histórico completo, vacinas, exames |
| **Agendamento** | Solicitar novas consultas |
| **Faturas** | Visualizar e pagar contas |
| **Prontuários** | Acessar históricos dos seus pets |
| **Exames** | Ver resultados de exames |
| **Receitas** | Visualizar prescrições |
| **Chat** | Conversar com a clínica |
| **Notificações** | Configurar seus canais de comunicação |

---

## Precisa de ajuda?

Entre em contato com a clínica pelo **Chat** disponível no portal ou ligue para a recepção.
