# Dashboard

O **Dashboard** é a página inicial do portal. Aqui você tem uma visão geral de tudo que precisa acompanhar.

## Cards do Dashboard

### Meus Pets
- Mostra quantos pets você tem cadastrados
- Clique no card para ver a lista completa
- De cada pet você pode acessar: histórico, vacinas, exames, prontuários

### Próximas Consultas
- Lista os agendamentos futuros
- Mostra: **nome do pet**, **data** e **horário**
- O **status** indica se a consulta está confirmada ou pendente
- Para cada consulta você pode:
  - **Confirmar** presença
  - **Reagendar** (dentro do prazo)
  - **Cancelar** (com até 24h de antecedência)

### Faturas Pendentes
- Número de contas em aberto
- Clique para ver detalhes e realizar pagamento

### Próximas Vacinas
- Mostra as vacinas agendadas para os próximos dias
- Exibe: **nome do pet**, **vacina**, **data prevista**
- Clique no card para ver o calendário completo de vacinas
- Vacinas com data vencida aparecem destacadas

### Widgets Visuais
O dashboard foi reformulado com cards maiores, ícones ampliados e cores por tipo:
- 🐾 **Pets** — verde
- 📅 **Consultas** — azul
- 💰 **Faturas** — amarelo
- 💉 **Vacinas** — roxo

## Atalhos Rápidos

- **Ver pets**: Clique no card "Pets" ou no menu superior
- **Nova consulta**: Vá em **Agendamento > Nova Consulta**
- **Ver faturas**: Clique no card "Faturas" ou no menu

## Dicas

- Os cards são atualizados em tempo real
- Use o menu no topo da página para navegar entre as seções
- Fique de olho no número de faturas pendentes para evitar atrasos
