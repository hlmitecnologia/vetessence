# Chat

## Visão Geral
O chat permite comunicação em tempo real entre tutores e a clínica, diretamente pelo sistema e pelo Portal do Tutor.

## Funcionalidades

### Mensagens
- Envio e recebimento de mensagens de texto
- Anexos: imagens, PDFs, resultados de exames
- Indicação de visualização (lida/não lida)
- Histórico completo de conversas

### Conversas
- Cada tutor tem uma conversa vinculada à clínica
- Identificação do tutor e dos pets associados
- Última mensagem em destaque
- Contador de mensagens não lidas

### Notificações
- Notificação na barra superior para novos chats
- Notificação sonora opcional
- Badge com número de mensagens não lidas

## Para o Tutor
1. Acesse o **Portal do Tutor**
2. Clique em **Chat**
3. Envie mensagens para a clínica
4. Anexe fotos e documentos
5. Receba respostas em tempo real

## Para a Clínica
1. Acesse **Comunicação > Chat**
2. Visualize conversas ativas na lista lateral
3. Clique em uma conversa para abrir
4. Responda mensagens
5. Encaminhe para o veterinário responsável

## Regras de Negócio
- Apenas tutores com pets cadastrados podem usar o chat
- Mensagens são mantidas por 90 dias
- Anexos têm limite de 10MB cada
- Chat não substitui canais de emergência
- Histórico pode ser exportado pelo tutor (LGPD)
